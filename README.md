# goruntuIsleme
Bu repository, Mustafa Ünlü tarafından verilen "OpenCV ile Görüntü İşleme" Udemy kursu süresince yazdığım ve öğrendiğim uygulama kodlarını içermektedir. Özellikle hızlı ilerleyebilmek için notlarımı defter yerine program üzerine yazdım. Bu konuda işin mantığına dayalı çok fazla Türkçe içerik bulunmadığından ve ilerleyen süreçlerde kendim de hızlı ulaşabilmek için notlarımı burada paylaşmak istedim. Bu içerikten faydalanabilmek için temel seviyede Python , Numpy kütüphanesi , Pandas Kütüphanesi , Matplotlib kütüphanesi ve OS kütüphanesi bilmek gerekir .
