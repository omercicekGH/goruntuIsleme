# goruntuIsleme
Bu repository, Mustafa Ünlü tarafından verilen "OpenCV ile Görüntü İşleme" Udemy kursu süresince yazdığım ve öğrendiğim uygulama kodlarını içermektedir. Özellikle hızlı ilerleyebilmek için notlarımı defter yerine program üzerine yazdım. Bu konuda mantığına dayalı çok fazla Türkçe içerik bulunmadığından notlarımı burada da paylaşmak istedim.
